## Project Overview

The Hospital Appointment Booking System is a web-based application developed using HTML, CSS, and JavaScript. The project allows patients to book appointments, view appointment records, search appointments, and manage appointment data through a simple and user-friendly interface.

The project demonstrates the implementation of Object-Oriented Programming (OOP) concepts and modular application design.

## How to Run

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Install the Live Server extension.
4. Run `index.html` using Live Server.
5. Use the application through the browser.
